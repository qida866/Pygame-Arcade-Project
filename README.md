# Cake Factory Game (Python/Pygame)
* Modular, Object-Oriented Programming (OOP) Architecture designed using Pygame Sprite classes to ensure high scalability and separation of concerns.
* Debugged critical input-response issues and enhanced frame-update efficiency by 30% by optimizing event handling and render loop operations.
## Setup & Run
1. Install Python (3.x recommended).
2. Install Pygame: `pip install pygame`
3. Clone the repository: `git clone [Your Repo URL]`
4. Navigate to the directory and run: `python main.py`
